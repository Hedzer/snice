⌨️ Keyboard Navigation

Try these keyboard shortcuts (focus on any tree first):

  • ↓ - Move to next visible item
  • ↑ - Move to previous visible item
  • → - Expand focused item (if it has children)
  • ← - Collapse focused item (if expanded)
  • Home - Move to first item
  • End - Move to last visible item
  • Enter or Space - Select focused item
