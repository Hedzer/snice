Camera Annotate Component

Capture images from camera and annotate them with drawings and labels.

Default Usage

Click "Capture" to take a photo, then draw on it. Add labels in the sidebar.

Features

  • Capture image from camera
  • Draw shapes/annotations on captured image
  • Label each annotation in sidebar
  • Hover label to highlight corresponding shape
  • Other shapes dim to opacity 0.2 + grayscale on hover
  • Show/hide individual annotation layers
  • Bulk show/hide all annotations
  • Auto-rotate colors through preset palette
  • Color picker with circular palette
  • Export annotated image with/without labels
  • Save/load annotation data (JSON)
  • Undo last stroke
  • Adjustable stroke width

Properties

  • mode - "camera" | "annotate" (default: "camera")
  • auto-rotate-colors - Boolean, cycle through palette colors (default: true)
  • show-labels-panel - Boolean, show sidebar (default: true)

Methods

  • capture() - Capture current frame and switch to annotate mode
  • exportImage({ includeLabels? }) - Export annotated image as data URL
  • exportAnnotations() - Export annotation data as JSON
  • importAnnotations(data) - Load annotation data
  • clearAnnotations() - Remove all annotations

Events

  • @snice/capture - Fired when image is captured
  • @snice/annotate - Fired when annotation is added
  • @snice/annotation-change - Fired when annotations change
