Highlighter Options

Code blocks support pluggable syntax highlighting:

  • Built-in - Simple highlighter included (JS, TS, HTML, CSS, Python, JSON, Bash)
  • Prism.js - See components/code-block/highlighters/prism.ts
  • highlight.js - See components/code-block/highlighters/highlight.ts
  • Custom - Implement your own HighlighterFunction

Example setup:

import { SniceCodeBlock } from 'snice/components/code-block';
import { highlightCode } from 'snice/components/code-block/highlighter';

// Use built-in highlighter globally
SniceCodeBlock.setGlobalHighlighter(highlightCode);

// Or per-instance
const codeBlock = document.querySelector('snice-code-block');
codeBlock.setHighlighter(highlightCode);
