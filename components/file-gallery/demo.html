File Gallery Component Demo

Basic File Gallery

Upload files with drag and drop support. Auto-upload is enabled by default.

Events:

Image Gallery

Restricted to image files only with preview thumbnails.

Manual Upload Mode

Auto-upload disabled. Files must be uploaded manually using the button.

File Limits

Maximum 3 files, each up to 2MB in size.

List View

Display files in a list layout instead of grid.

Custom Options

Gallery with pause/resume disabled and custom accept types.

Add Button Mode

Instead of a drop zone, shows an add button tile in the gallery grid. Click the plus tile to add files.

Files with Avatar Badges

Files can have custom badges overlaid on thumbnails (top-left, top-right, bottom-left, bottom-right). Perfect for showing user avatars on collaborative files.

Custom Actions with Badges

Use addCustomAction(icon, text) to add custom buttons. Files get badges showing their source.
