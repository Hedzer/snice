Features

  • Drag and drop cards between columns and within the same column
  • Click on cards to view details
  • Card labels with custom colors (text + background)
  • Card assignees
  • Color-coded cards and columns
  • Card counts per column
  • Search cards by title or description
  • Filter by labels - can select multiple labels (AND logic)
  • View Transitions API - smooth animations when moving cards

API Usage

Try these in the browser console:

// Search for cards
kanban.search('landing');

// Filter by labels (show cards with these labels)
kanban.filterByLabels(['design', 'high-priority']);

// Clear all filters
kanban.clearFilters();

// Add card with colored labels
kanban.addCard('todo', {
  id: 99,
  title: 'New task',
  labels: [
    { text: 'urgent', color: '#dc2626', background: '#fee2e2' },
    { text: 'backend', color: '#9333ea', background: '#f3e8ff' }
  ]
});

// String labels still work (will use default styling)
kanban.addCard('todo', {
  id: 100,
  title: 'Another task',
  labels: ['feature', 'ui']
});
