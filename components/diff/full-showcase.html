Component Properties

  • old-text - The original text content
  • new-text - The modified text content
  • mode - Display mode: 'unified' (default) or 'split'
  • language - Optional language hint for syntax highlighting
  • line-numbers - Show line numbers (default: true)
  • context-lines - Number of context lines around changes (default: 3)
  • markers - Show +/- markers (default: true)

Events

The component dispatches a diff-computed event when the diff is calculated:

diff.addEventListener('diff-computed', (e) => {
  console.log(e.detail.hunks);      // Array of diff hunks
  console.log(e.detail.additions);  // Number of lines added
  console.log(e.detail.deletions);  // Number of lines removed
});
