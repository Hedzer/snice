What is Snice?

Snice is a modern web component library that provides customizable, accessible UI components built with TypeScript and decorators. It focuses on simplicity, performance, and developer experience.

 How do I install Snice?

You can install Snice using npm:

npm install snice

Then import the components you need in your application.

 Is Snice accessible?

Yes! All Snice components are built with accessibility in mind. They include proper ARIA attributes, keyboard navigation, and follow WCAG guidelines.

  • Full keyboard support
  • Screen reader friendly
  • ARIA compliant
  • Focus management

 Code Example 
import { SniceAccordion } from 'snice/accordion';

// Get accordion element
const accordion = document.querySelector('snice-accordion');

// Open specific item
accordion.openItem('item-1');

// Listen for events
accordion.addEventListener('accordion-open', (e) => {
  console.log('Opened:', e.detail.itemId);
});
