Features

  • Auto-starts camera feed on load
  • Subtle Material Design-style icons
  • Capture button with camera icon (56px)
  • Switch camera button (40px, auto-hides if only one camera)
  • Corner positioning: bottom-left, bottom-right, top-left, top-right
  • Edge positioning: bottom, right, left, top
  • Auto-detect orientation (bottom-right for portrait, right for landscape)
  • Slotted custom controls overlay
  • Configurable resolution (default 720p HD)
  • Aspect ratio presets (16:9, 4:3, 1:1, 21:9)
  • Fullscreen mode support
  • Mirror mode for front camera

Usage

<snice-camera></snice-camera>

Zero config - auto-starts with bottom-right controls in portrait mode.

Attributes

  • controls-position - "auto" (default), corners: "bottom-left", "bottom-right", "top-left", "top-right", edges: "bottom", "right", "left", "top"
  • show-controls - Boolean, show built-in controls (default: true). Set to false to use only custom slotted controls
  • width - Number, video width in pixels (default: 1280)
  • height - Number, video height in pixels (default: 720)
  • aspect-ratio - String, aspect ratio preset: "auto" (default, expands to fill space), "16:9", "9:16", "4:3", "1:1", "21:9"
  • object-fit - "cover" (default, fills frame) or "contain" (shows full video, may letterbox)
  • facing-mode - "user" (front, default) or "environment" (back)
  • mirror - Boolean, mirrors video for front camera (default: true)
  • auto-start - Boolean, auto-start on load (default: true)

Methods

  • capture() - Capture current frame as image
  • switchCamera() - Toggle between front/back camera
  • toggleFullscreen() - Toggle fullscreen mode
  • enterFullscreen() - Enter fullscreen mode
  • exitFullscreen() - Exit fullscreen mode

Slots

  • controls - Custom controls overlay area (full viewport positioned absolutely)
