Features

  • Multiple chart types (line, bar, area, pie, donut, scatter, bubble, radar, mixed)
  • Interactive legend with dataset toggling
  • Tooltips on hover
  • Customizable colors and styles
  • Responsive and animated
  • Multiple datasets support
  • SVG export capability
  • Grid and axis customization

Supported Chart Types

  • ✓ Line Chart - Trends over time
  • ✓ Bar Chart - Categorical comparisons
  • ✓ Horizontal Bar - Alternative bar orientation
  • ✓ Area Chart - Filled line charts
  • ✓ Pie Chart - Part-to-whole relationships
  • ✓ Donut Chart - Pie with center hole
  • ✓ Scatter Plot - Point distributions
  • ✓ Bubble Chart - 3-dimensional data
  • ✓ Radar Chart - Multivariate data
  • ✓ Mixed/Combo - Multiple chart types combined
