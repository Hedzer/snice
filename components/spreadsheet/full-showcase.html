Spreadsheet Component Demo

Editable spreadsheet with cell selection, formulas, sorting, undo/redo, clipboard support, and context menu.

Basic Spreadsheet

Click a cell to select, double-click to edit, type to start editing

Keyboard: Arrow keys to navigate, Enter/F2 to edit, Tab to move right, Shift+Tab to move left, Ctrl+Z/Y for undo/redo, Delete to clear. Shift+click or Shift+Arrow for multi-select. Right-click for context menu.

With Typed Columns

Number, text, boolean, and select column types

Formulas

SUM, AVG, COUNT, MIN, MAX with cell ranges

Try editing cells A1-A5 and watch the formula results update. Supported: =SUM(A1:A5), =AVG(A1:A5), =COUNT(A1:A5), =MIN(A1:A5), =MAX(A1:A5)
