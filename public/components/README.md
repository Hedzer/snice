# Snice Time Picker - CDN Build

CDN build of the Snice time-picker component. Requires the Snice runtime (`snice-runtime.min.js`) to be loaded first.

## Usage

```html
<script src="snice-runtime.min.js"></script>
<script src="snice-time-picker.min.js"></script>
<snice-time-picker></snice-time-picker>
```

## Size
- Minified: 27.6 KB
- Gzipped: 7.0 KB

## Theme
For proper styling, link the Snice theme CSS:
```html
<link rel="stylesheet" href="https://unpkg.com/snice/dist/components/theme/theme.css">
```
