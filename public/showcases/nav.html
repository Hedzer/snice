Nav

Application navigation with flat, hierarchical, and grouped variants in horizontal or vertical orientations

Horizontal (flat)
Hierarchical
Grouped
javascript 
// Flat navigation
nav.update(placards, appContext, 'dashboard');

// Hierarchical (parent-child)
nav.update([
  { name: 'products', title: 'Products', icon: '\ud83d\udce6' },
  { name: 'products/list', parent: 'products',
    title: 'Product List' },
], ctx, 'products/list');

// Grouped (by category)
nav.update([
  { name: 'users', title: 'Users',
    group: 'Management', icon: '\ud83d\udc65' },
], ctx, 'settings');

// Variants: flat, hierarchical, grouped
// Orientation: horizontal, vertical
