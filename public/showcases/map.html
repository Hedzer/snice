Map

Interactive tile-based map with markers, popups, and zoom controls

html <snice-map zoom="13"></snice-map> <script> const map = document.querySelector( 'snice-map'); map.center = { lat: 51.505, lng: -0.09 }; map.markers = [ { id: '1', lat: 51.505, lng: -0.09, label: 'London', popup: 'Hello London!' } ]; map.addEventListener('marker-click', e => console.log(e.detail.marker)); map.fitBounds(); </script>
