Standalone Build

Use any component without the full framework. Each element includes a full copy of Snice.

npx snice build-component button
# creates snice-button.min.js (~25kb gzipped)

<script src="snice-button.min.js"></script>
<snice-button variant="primary">Works anywhere</snice-button>

Using 3+ components? Load the runtime once and use light builds (~2-5KB each). See the Runtime + Light guide for details.
