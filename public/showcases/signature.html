Signature

Canvas-based signature pad with smooth bezier curves and PNG export

Clear
html 
<snice-signature
  stroke-color="rgb(0 0 128)"
  stroke-width="3"
></snice-signature>

<script>
  const sig = document.querySelector(
    'snice-signature');

  // Check if signed
  console.log(sig.isEmpty());

  // Export as data URL or Blob
  const dataUrl = sig.toDataURL();
  const blob = await sig.toBlob();

  // Clear pad
  sig.clear();
</script>
