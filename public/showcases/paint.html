Draw + Paint

Self-contained drawing canvas with built-in toolbar. Uses <snice-draw> under the hood — use it standalone for a bare canvas with your own UI.

html <!-- All-in-one with built-in toolbar --> <snice-paint></snice-paint> <!-- Custom colors & controls --> <snice-paint colors='["#ff0000","#00ff00","#0000ff"]' controls="colors,size,undo,redo" ></snice-paint> <!-- Bare canvas only (bring your own toolbar) --> <snice-draw width="800" height="600" color="#3b82f6" stroke-width="3" ></snice-draw> <script> // Paint: self-contained const paint = document.querySelector('snice-paint'); paint.download('artwork.png'); // Draw: bare canvas, wire up your own UI const draw = document.querySelector('snice-draw'); draw.color = '#ef4444'; draw.undo(); draw.clear(); </script>
